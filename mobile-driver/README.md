# YathraGo Driver App

This is the **YathraGo Driver** mobile application built with [Expo](https://expo.dev) and React Native. This app empowers drivers to receive ride requests, navigate efficiently, and manage their trips.

## ✨ Features

- **Approve booking Requests**: Receive and accept vehicle bookings with customization
- **GPS Navigation**: Built-in map integration for optimized routing.
- **Trip Management**: Track daily schedules and completed trips.
- **Cross-platform**: Works seamlessly on both Android and iOS devices.

## 🛠️ Tech Stack

- **Framework**: React Native, Expo
- **Styling**: NativeWind (Tailwind CSS)
- **Language**: TypeScript
- **Navigation**: Expo Router (File-based routing)

## 🚀 Getting Started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**

   ```bash
   npm run dev
   # or
   npx expo start
   ```

3. **Run on a device or emulator**
   - Use Android Studio Emulator for the best development experience.
   - Or scan the QR code with Expo Go on your physical device.
