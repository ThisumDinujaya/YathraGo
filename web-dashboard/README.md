# YathraGo Web Dashboard

This is the administrative web dashboard for the YathraGo transport management system, built with [Next.js](https://nextjs.org/) and React.

## 🚀 Features

- **Admin Portal**: Manage users, drivers, and vehicles across the platform.
- **Vehicle Owner Dashboard**: Track vehicle performance, manage drivers, and view earnings.
- **Transport Manager View**: Oversee daily operations, routes, and scheduling.
- **Modern UI**: Clean and responsive interface built with Tailwind CSS.

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Library**: React.js
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## ⚙️ Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the dashboard.

## 📁 Project Structure

The project uses the Next.js App Router paradigm:
- `app/`: Contains the application routes, pages, and layouts.
- `components/`: Reusable React components.
- `public/`: Static assets like images and icons.
